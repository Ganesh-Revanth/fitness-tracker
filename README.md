# Fitness Tracker

A fitness tracker web application built to help users track their workouts and fitness progress.

## Features
- Responsive Landing Page
- User Login Interface
- User Registration Interface
- Clean and Modern UI

## Tech Stack
- HTML
- CSS
- JavaScript (Learning)
- Firebase Authentication
- Firebase Realtime Database

## Project Status
🚧 This project is currently under development.

The landing page, login page, and registration page with authentication are complete.

I'm currently working on the dashboard and implementing its functionality.
